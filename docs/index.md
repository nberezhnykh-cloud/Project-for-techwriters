## Что здесь есть:

## Как все организовано:

Сайт собран с помощью [mkdocs.org](https://www.mkdocs.org).

### Полезные команды для MkDocs:

* `mkdocs new [dir-name]` - создание нового проекта.

* `mkdocs serve` — локальная сборка для просмотра сайта.

* `mkdocs build` – финальная сборка сайта.

* `mkdocs -h` - вывод справочного сообщения и остановка работы.
