# ⌨️🔣 Alt-коды и сочетания клавиш для ввода спецсимволов

При оформлении документов часто требуются **корректные знаки** – например, кавычки-«елочки», длинное тире (—) или неразрывный пробел. Однако в таких приложениях, как **Microsoft Excel**, **Confluence** функция автоматической замены (например, превращение обычных кавычек в «елочки») либо отключена, либо отсутствует полностью.

В этих случаях надежным способом вставить нужный символ является использование **Alt-кодов** – числовых комбинаций, которые работают в большинстве Windows-приложений при наличии цифровой клавиатуры (NumPad) или сочетаний клавиш на macOS.

## 🪟 Windows

Ниже приведён справочник часто используемых символов и соответствующих им Alt-кодов.

***
!!! warning "Важно!"
    Коды вводятся только на цифровой клавиатуре (NumPad) при включенном Num Lock.

    Удерживайте клавишу Alt, наберите код (например, 0171), затем отпустите Alt – появится нужный символ (в данном случае – «).
***

## 🍎 macOS

***
!!! note "Примечание"
    Для набора специальных символов на macOS цифровая клавиатура (NumPad) вам не нужна.
    Приведённые в таблице цифры – это обычные цифры, расположенные на числовом ряде клавиатуры.

    Для некоторых символов (например, «↑», «↓», «←», «→») нет стандартного сочетания клавиш, поэтому при необходимости их вставки можно использовать другие методы:

    * **панель «Эмодзи и символы»**: откройте её через меню «Правка» → «Эмодзи и символы» или с помощью сочетания клавиш **⌃ Control + ⌘ Command + Space** в любой программе для ввода текста;

    * **шестнадцатеричный ввод в Юникоде**. Если в системных настройках клавиатуры включён этот режим, можно, удерживая клавишу ⌥ Option, ввести четырёхзначный шестнадцатеричный код символа (например, U+2191 для стрелки вверх);

    * **⌘ Command + C**, **⌘ Command + V** в случае разовой необходимости.
***
