# 📝 Язык разметки Markdown
## ℹ️ Общая информация
### ❓ Что такое Markdown

Markdown (маркдаун) — это облегчённый язык разметки для форматирования текстов.

Для разметки в маркдауне используются знаки препинания и специальные символы, разные комбинации которых позволяют выделять и структурировать текст.

Размеченный с помощью маркдауна текст легко читается в исходном виде, а ещё он без проблем конвертируется в другие форматы, например, `html` или `pdf`.

### 📐 Синтаксис

Базовый синтаксис маркдауна очень прост и поддерживает основные элементы структурирования и форматирования текста: заголовки разных уровней, абзацы, маркированные и нумерованные списки, цитаты, ссылки и изображения, жирное начертание и курсив.

Базовый синтаксис описан в стандартной спецификации CommonMark.

#### 📋 Спецификация CommonMark

Спецификация CommonMark была создана сообществом для того, чтобы чётко определить его синтаксис и создать стандартную, совместимую и тестируемую версию языка.

## 📖 Шпаргалка по синтаксису
### 🟦 Базовый синтаксис Markdown
#### 📌 Заголовки

В синтаксисе Markdown есть шесть уровней заголовков: от H1 (самого большого) до H6 (самого маленького). Для их выделения используют решётки `#`:

* решётки можно ставить как перед заголовком, так и с двух сторон от него (на уровень заголовка влияют только те `#`, которые находятся перед ним);

* количество решёток соответствует уровню заголовка: одна для первого уровня, две для второго и так далее;

* между решёткой и текстом ставится пробел.

***
!!! info "Информация"
    У заголовков первого и второго уровня есть альтернативный способ выделения: на следующей строке после них нужно поставить знаки равенства = или дефисы -. Вот несколько правил:
    
    * знак = применяется для заголовков H1;
    
    * дефис применяется для заголовков H2;
    
    * если в одной строке поставить оба знака, то работать ничего не будет;
    
    * можно ставить любое количество знаков, и на тип заголовка это не повлияет;
    
    * между заголовком и знаками не должно быть пустых строк.
***

```
# Заголовок 1 уровня
## Заголовок 2 уровня
### Заголовок 3 уровня
#### Заголовок 4 уровня
##### Заголовок 5 уровня
###### Заголовок 6 уровня
```

!!! quote ""
    <h1>Заголовок 1 уровня</h1>

!!! quote ""
    <h2>Заголовок 2 уровня</h2>


!!! quote ""
    <h3>Заголовок 3 уровня</h3>


!!! quote ""
    <h4>Заголовок 4 уровня</h4>


!!! quote ""
    <h5>Заголовок 5 уровня</h5>


!!! quote ""
    <h6>Заголовок 6 уровня</h6>

##### ✨ Выделение текста
###### *Курсив (italic)*

```
*Курсив*
_Курсив_
```

!!! quote ""
    *Курсив*

    _Курсив_

###### **Жирный (bold)**

```
**Жирный шрифт**
__Жирный шрифт__
```

!!! quote ""
    **Жирный шрифт**

    __Жирный шрифт__

###### ***Жирный курсив (bold and italic)***

```
***Текст жирным курсивом***
___Текст жирным курсивом___
```

!!! quote ""
    ***Текст жирным курсивом***

    ___Текст жирным курсивом___

***
!!! warning "Важно"
    Если вы хотите выделить фрагмент внутри слова *курсивом* или **жирным**, то это корректно сработает только при использовании звёздочек:
    
    ```
    кор*рек*тно, кор**рек**тно
    некор_рек_тно, некор__рек__тно
    ```

    !!! quote ""
        кор*рек*тно, кор**рек**тно

        некор_рек_тно, некор__рек__тно


    В случае, если нужно выделить фрагмент внутри слова ***жирным курсивом***, то можно использовать оба варианта – три звёздочки *** или три нижних подчёркивания ___:

    ```
    кор***рек***тно
    кор___рек___тно
    ```

    !!! quote ""
        кор***рек***тно

        кор___рек___тно
***

###### <u>Подчёркнутый (underline)</u>

В синтаксисе Markdown нет встроенного способа подчеркнуть текст.

Но если ваш редактор поддерживает HTML, то можно использовать теги:

```
<u>Подчёркнутый текст</u>
```

!!! quote ""
    <u>Подчёркнутый текст</u>

###### <s>Зачёркнутый (strikethrough)</s>

В синтаксисе Markdown нет встроенного способа зачеркнуть текст.

Но если редактор поддерживает HTML, то можно использовать теги:

```
<s>Зачёркнутый текст</s>
```

!!! quote ""
    <s>Зачёркнутый текст</s>

###### 🎨 Цветной шрифт (Text Color & Font Color)

В синтаксисе Markdown нет встроенного способа менять цвет текста.

Но если редактор поддерживает HTML, то можно использовать теги:

```
<span style="color: red;">Красный текст</span>
```

!!! quote ""
    <span style="color: red;">Красный текст</span>

***
!!! note "Примечание"
    Можно использовать названия цветов, HEX-коды (#FF0000) или RGB.

    Полезный сайт, на котором представлены поодерживаемые всеми браузерами цвета [тут 🔗](https://www.w3schools.com/colors/colors_names.asp). На каждый цвет можно нажать и посмотреть его название, HEX-код и RGB, а также как он сочетается с другими цветами.

    ```
    <span style="color:red">Красный текст</span>
    <span style="color:#0000FF">Синий текст</span>
    <span style="color:rgb(0,128,0)">Зеленый текст</span>
    ```

    !!! quote ""
        <span style="color:red">Красный текст</span>

        <span style="color:#0000FF">Синий текст</span>

        <span style="color:rgb(0,128,0)">Зеленый текст</span>
***    

##### ➖ Разделители

Чтобы оформить горизонтальный разделитель, нужно поставить три или больше специальных символа: звёздочки `*`, дефиса `-` или нижних подчёркивания `_`.

Если редактор поддерживает HTML, то также можно использовать тег `<hr>`.

Они должны находиться на отдельной строке, и между ними можно ставить любое количество пробелов и табуляций.

```
***
---
___
**	*  **
<hr>
```

!!! quote ""
    ***

    ---

    ___

    **	*  **

    <hr>

```

```

!!! quote ""
    >Цитата

#### 📋 Списки

В синтаксисе Markdown есть несколько видов списков.

Для их оформления перед каждым пунктом нужно поставить подходящий тег (цифра, буллет) и отделить его от текста пробелом.

##### 🔢 Простой список
###### 1️⃣ Нумерованный список

Для создания нумерованного списка перед пунктами нужно поставить число с точкой.
При этом неважно, какие именно числа будут написаны: Markdown пронумерует список автоматически.

```
1. Пункт один
2. Пункт два
3. Пункт три
```

!!! quote ""
    1. Пункт один
    2. Пункт два
    3. Пункт три

```
1. Пункт один
1. Пункт два
1. Пункт три
```

!!! quote ""
    1. Пункт один
    1. Пункт два
    1. Пункт три

```
1. Пункт один
88. Пункт два
6. Пункт три
```

!!! quote ""
    1. Пункт один
    88. Пункт два
    6. Пункт три

###### 🔘 Ненумерованный список

Для создания ненумерованного списка нужно поставить перед каждым пунктом звёздочку `*`, дефис `-` или плюс `+`.

```
* Пункт с буллетом
* Пункт с буллетом
* Пункт с буллетом
```

!!! quote ""
    * Пункт с буллетом
    * Пункт с буллетом
    * Пункт с буллетом

```
- Пункт с буллетом
- Пункт с буллетом
- Пункт с буллетом
```

!!! quote ""
    - Пункт с буллетом
    - Пункт с буллетом
    - Пункт с буллетом

```
+ Пункт с буллетом
+ Пункт с буллетом
+ Пункт с буллетом
```

!!! quote ""
    + Пункт с буллетом
    + Пункт с буллетом
    + Пункт с буллетом

##### 📂 Вложенный список

Для создания вложенного списка нужно поставить перед его пунктами табуляцию или несколько пробелов. В Markdown одна табуляция соответствует четырём пробелам.

Список одного вида можно вкладывать в другой.

```
1. Пункт
	1. Подпункт
		1. Подподпункт

- Пункт
	* Подпункт
		1. Подподпункт

+ Пункт
	1. Подпункт
```

!!! quote ""
    1. Пункт
        1. Подпункт
            1. Подподпункт

    - Пункт
        * Подпункт
            1. Подподпункт

    + Пункт
        1. Подпункт

***
!!! warning "Важно"
    Необходимо учитывать, что теги `*`и `-` на 2 и 3 уровне создают буллет `•`.

    ```
    1. Пункт
        - Подпункт
            * Подподпункт
    ```

    !!! quote ""
        1. Пункт
            - Подпункт
                * Подподпункт
***

#### 💻 Код
##### ⌨️ Встроенный код

Чтобы оформить часть текста как код, используется такой синтаксис:

```
`Hello, World!`
```

!!! quote ""
    `Hello, World!`

##### 📦 Блок кода

Для оформления части текста как блока кода можно использовать несколько вариантов:

* Три обратных апострофа (```):

!!! quote ""
    ```
    printf("Hello, World!")
    ```

* Двойная табуляция (восемь пробелов):

!!! quote ""
        printf("Hello, World!")

***
!!! note "Подсветка синтаксиса"
    Если обрамлять код тремя обратными апострофами, то после первой тройки можно указать язык программирования – тогда Markdown подсветит элементы кода:

    === "С подсветкой"

        ```py
        package main

        import "fmt"

        func main() {
            fmt.Println("Hello, World!")
        }
        ```

    === "Без подсветки"

        ```
        package main

        import "fmt"

        func main() {
            fmt.Println("Hello, World!")
        }
        ```
***

#### 🔗 Ссылка

Чтобы оформить ссылкой часть текста, используется такой синтаксис: `[текст](ссылка)`. 

```
[ссылка 🔗](http://news.yandex.ru)
```

!!! quote ""
    [ссылка 🔗](http://news.yandex.ru)

Чтобы добавить внутреннюю ссылку на страницу сайта, в круглых скобках нужно указать полное название файла.

```
[ссылка 🔗](onboarding.md)
```

!!! quote ""
    [ссылка 🔗](onboarding.md)
***
!!! note "Добавление подсказки в ссылку"
    К ссылке можно добавить всплывающую подсказку при наведении курсора. Для этого в круглых скобках после ссылки нужно поставить пробел и написать текст подсказки в кавычках.

    ```
    [ссылка 🔗](onboarding.md "Переход на страницу OnBoarding")
    ```

    !!! quote ""
        [ссылка 🔗](onboarding.md "Переход на страницу OnBoarding")
***

#### 📊 Таблица

Таблицы создаются с помощью вертикальных знаков `|` и дефисов `-`.

Столбцы разделяются вертикальными линиями `|`, а строка с шапкой отделяется от остальных дефисами `-`, которых можно ставить сколько угодно.

```
|Столбец 1|Столбец 2|Столбец 3|
|-|--------|---|
|Длинная запись в первом столбце|Запись в столбце 2|Запись в столбце 3|
|Кртк зпс| |Слева нет записи|
```

!!! quote ""
    |Столбец 1|Столбец 2|Столбец 3|
    |-|--------|---|
    |Длинная запись в первом столбце|Запись в столбце 2|Запись в столбце 3|
    |Кртк зпс| |Слева нет записи|

***
!!! note "Выравнивание текста"
    Чтобы выровнять весь столбец по правому или левому краю, в строке с дефисами  после или перед дефисом надо поставить двоеточие `:`. 
    
    Чтобы выровнять содержимое по центру, надо поставить `:` с обеих сторон от дефиса.

    ```
    |Столбец 1|Столбец 2|Столбец 3|
    |:-|:-:|-:|
    |Выравнивание по левому краю|Выравнивание по центру|Выравнивание по правому краю|
    |Запись|Запись|Запись|
    ```

    !!! quote ""
        |Столбец 1|Столбец 2|Столбец 3|
        |:-|:-:|-:|
        |Выравнивание по левому краю|Выравнивание по центру|Выравнивание по правому краю|
        |Запись|Запись|Запись|
***

### 🚀 Расширенный синтаксис Markdown
#### ✍🏻 Примечание

Чтобы оформить блок текста как примечание, используется такой синтаксис:

```
!!! note "Примечание"
    Текст примечания
```
***
!!! note "Примечание"
    Текст примечания
***

#### ℹ️ Информация

Чтобы оформить блок текста как информационную справку, используется такой синтаксис:

```
!!! info "Информация"
    Текст примечания
```
***
!!! info "Информация"
    Текст примечания
***

#### 💬 Цитата

Чтобы оформить блок текста как цитату, используется такой синтаксис:

```
!!! quote "Цитата"
    Текст примечания
```

***
!!! quote "Цитата"
    Текст примечания
***

#### ⚠️ Внимание

Чтобы оформить блок текста как предупреждение используется такой синтаксис:

```
!!! warning "Внимание"

```

***
!!! warning "Внимание"
    Текст примечания
***
